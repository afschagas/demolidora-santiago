# demolidora-santiago

Site institucional da **Demolidora Santiago** — React, Vite, TypeScript e Tailwind CSS, com tema claro/escuro.

## Scripts

- `npm run dev` — servidor de desenvolvimento (rede local: `vite.config` com `host: true`)
- `npm run build` — build de produção (`dist/`)
- `npm run preview` — pré-visualização do build

## Stack

- React 18 + React Router
- Vite 5
- Tailwind CSS 3

## Deploy

Após `npm run build`, publique o conteúdo da pasta `dist/` no seu provedor (ex.: GitHub Pages, Netlify, Vercel).
